Gym Management App

Welcome to your gym management app.

This app was built using ASP.NET Core, Angular2, Bootstrap and more!

You can now manage your gym from everywhere with your pc! You only need an Internet connection

First of all, create your personal database: it will include training rooms, classes, users, training programs, workout equipment and trainers. You can choose to create an empty database or a database already populated with some elements.

You can manage the entities of the database by using the menu on the left.

In every single page you will find tables and buttons to act on those entities: read, create, update and delete.

Enjoy this new web app!



Use those buttons to create or delete your database:

  • Create database (not populated)
  • Create database (populated)
  • Drop database
